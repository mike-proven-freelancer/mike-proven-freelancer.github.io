Project 03 - files and how to reproduce

Paths are relative to the repository root.

What lives where

  • DA-Prediction/generate_prediction_assets.py - generates source data, metrics, charts, and forecast outputs.
  • DA-Prediction/data/inventory_sales_source.xlsx - single source file (36 months x monthly columns).
  • DA-Prediction/data/inventory_sales_source.csv - CSV mirror for easy inspection.
  • DA-Prediction/data/forecast_2_months.csv - two-month forecast and scenario values.
  • DA-Prediction/data/insights.json - quick summary values used for narrative checks.
  • DA-Prediction/images/charts/*.png - figures used in project and blog pages.

Main monthly columns

Examples (for each brand):

  • a_brand_inventory, a_brand_sales, a_brand_pct_sold
  • a_brand_pct_sold_mom_change_pp, a_brand_pct_sold_yoy_change_pp
  • a_brand_inventory_mom_change_pct, a_brand_inventory_yoy_change_pct
  • a_brand_sales_mom_change_pct, a_brand_sales_yoy_change_pct

The same columns exist for b_brand and c_brand, plus totals like total_inventory, total_sales, and total_pct_sold.

Run steps (Windows)

py -m pip install -r DA-Prediction\requirements.txt
py DA-Prediction\generate_prediction_assets.py

Re-running updates the dataset window to the latest 36 months up to the current month.
