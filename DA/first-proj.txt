Summary

We are looking for a strong data scientist or analytical researcher to help build and maintain structured datasets used to evaluate predictive decision models.

This project involves combining information from multiple public sources and structuring it into research-grade datasets that can be used to test analytical frameworks and predictive systems.

The work sits at the intersection of:

• data science
• research and pattern analysis
• model evaluation
• structured dataset design

This is not a typical machine learning task with clean data. Much of the work involves organizing messy real-world information into structured formats that can be used for analysis.

Responsibilities

You will help with:

• Collecting and structuring datasets from public sources
• Reconstructing historical information snapshots
• Designing data schemas for analytical models
• Running backtests and comparing model outputs to historical outcomes
• Identifying patterns across structured datasets
• Improving evaluation frameworks over time

Some work may involve combining traditional data science methods with newer AI-based analysis tools.

Ideal Background

We’re looking for someone who is comfortable working with messy datasets and ambiguous problems.

Strong candidates typically have experience with:

• Python and data analysis libraries
• statistical analysis or machine learning
• dataset construction and cleaning
• experimental design or model evaluation
• working with mixed structured and unstructured data

Experience with research workflows or analytical frameworks is a plus.

Nice to Have

• experience with LLM-based workflows
• experience building benchmark datasets
• familiarity with historical data research
• strong written documentation skills

Project Scope

Initial project:

• help design and structure a benchmark dataset
• build data pipelines and schemas
• assist with model evaluation experiments

If the collaboration goes well, this may expand into ongoing work improving the research system.
