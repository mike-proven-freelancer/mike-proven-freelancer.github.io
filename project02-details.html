Project 02 — Files & reproduction

Paths are relative to the repository root (this portfolio repo).

What lives where

  • DA-COBOL/cobol/bank_main.cob — main COBOL program; writes DA-COBOL/data/transactions.csv.
  • DA-COBOL/cobol/hello.cob — tiny compile check.
  • DA-COBOL/analysis/spending_analysis.py — loads CSV, prints summaries, saves PNGs under DA-COBOL/images/charts/.
  • DA-COBOL/requirements.txt — Python dependencies (pandas, matplotlib).
  • DA-COBOL/README.md — short local notes for the folder.

CSV shape (no header row)

Columns in order: txn_id, account_id, txn_date, txn_type, amount, balance_after, merchant, category. String fields may be padded with spaces from COBOL; the Python script strips them.

COBOL (Windows)

From DA-COBOL:

cobc -x cobol\bank_main.cob -o bank_main.exe
.\bank_main.exe

Compile output is bank_main.exe in DA-COBOL (ignored by git if listed in .gitignore).

Python

From DA-COBOL:

py -m pip install -r requirements.txt
py analysis\spending_analysis.py

Charts are written to DA-COBOL/images/charts/ when matplotlib is available.
